## Overview

Poj Pro Operations Platform — OOO «Poj Pro» uchun production ichki biznes tizimi. .NET MAUI desktop klienti Railway dagi ASP.NET Core API ga ulanadi: sotuvlar, ombor, moliya, shartnomalar, menejer KPI. Klient online/offline rejimini avtomatik aniqlaydi va SQLite navbatini sync qiladi. Holat: Production.

Repozitoriy: [WinstonSalemm/projectApp](https://github.com/WinstonSalemm/projectApp). Ommaviy katalog — [`poj-pro-site`](https://www.poj-pro.uz) ([Poj-Pro](https://github.com/WinstonSalemm/Poj-Pro)). Shartnomalar va botlar — `poj-pro-api-contracts`, `poj-pro-telegram-bots`.

## Problem Statement

Sotuv, ombor va moliya jarayonlari jadvallar va messenjerlarda tarqalgan edi. Naqd/karta sotuvlar, bron, qarzlar, ND-40/IM-40 yetkazib berish, soliq analitikasi (UZ) va shartnomalar uchun yagona tizim kerak — internet vaqtincha uzilganda ham ishlashi kerak.

## Business Context

Foydalanuvchilar — ichki operatorlar, menejerlar va ombor xodimlari. Yuk Tashkent ish vaqtida yuqori. Desktop — kundalik operatsion interfeys; https://www.poj-pro.uz B2B xaridorlar uchun alohida.

## Technology Stack

| Layer      | Technology          | Role                                 |
| ---------- | ------------------- | ------------------------------------ |
| Client     | .NET MAUI, .NET 8   | Windows desktop UI, offline queue    |
| API        | ASP.NET Core        | Business logic, REST, Railway deploy |
| Offline    | SQLite              | Lokal saqlash va operatsiya navbati  |
| Production | Railway, PostgreSQL | Hosted API va asosiy DB              |

## Trade-offs

**Desktop MAUI vs web admin.** Desktop — tez kiritish va offline ishonchlilik.

**Monolith API.** Bitta deploy sync va tranzaksiyalarni soddalashtiradi.

**Alohida public site.** poj-pro.uz katalogi mustaqil deploy.

## Roadmap

**Maintained.** Sotuv va ombor, offline sync ishonchliligi.

**Planned.** `poj-pro-api-contracts` bilan chuqurroq integratsiya.

**Deferred.** Windows dan tashqari MAUI targetlar.

## Interface Record

Faqat Windows desktop: sotuvlar, ombor, mijozlar, shartnomalar, moliya, soliq analitikasi, KPI. Sync indikatori — yuqori o'ng burchak.

## Lessons Recorded

**Sync navbati ko'rinishi kerak.** Pending operatsiya soni ishonchni oshiradi.

**Public web va internal ops — alohida mahsulotlar.** `Poj-Pro` va `projectApp` ajratilgan.

**API URL — muhit konfigi.** Railway base URL `appsettings.json` da.
